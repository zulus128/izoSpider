# iZoSpider

Early-stage iOS application and game prototype developed using Objective-C.

## 📱 Overview

iZoSpider is an early mobile project that demonstrates foundational experience in iOS development, game mechanics, and UI interaction.

The project was built as part of hands-on exploration of mobile application architecture and game logic implementation.

## 🚀 Features

- Custom UI components and interactions  
- Basic game mechanics and logic  
- Asset management (sprites, tiles, UI elements)  
- Native iOS development using Objective-C  

## 🛠 Tech Stack

- Objective-C  
- Xcode  
- iOS SDK  

## 👨‍💻 Role

- Full-cycle development  
- UI implementation  
- Game logic and interaction design
